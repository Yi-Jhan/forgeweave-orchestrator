# ForgeWeave Orchestrator 參考文件

本資料夾保存完整設計參考、歷史藍圖、架構脈絡與長期方向。

這些文件不是 Codex 自動實作的主要依據。Codex 進行 Phase Runner 或一般實作時，應優先遵守：

1. `docs/planning/execution-charter.md`
2. `docs/tasks/phase-gates.md`
3. `docs/tasks/task-list.md`
4. `docs/specs/spec.md`
5. `docs/roadmap/blueprint.md`

不得因為 reference 文件提到未來能力，就提前實作目前 Phase 未要求的功能。若 reference 文件與目前 task、phase gate、spec 或簡化版 blueprint 衝突，以目前執行文件為準。
